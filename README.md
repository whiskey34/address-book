<p align="center">ADDRESS BOOK</p>

## FEATURE

<p align="center">
- Have CRUD  operations for contacts.
- Show all the contact list in a table format
- Search and filter by name, email or phone number
- Single Page Application (SPA) 
</p>

## HOW TO RUN APP

-   Clone this repository to your local machine using `git clone https://github.com/whiskey34/address-book` command.
-   Extract folder to your local machine
-   using npm install command to get started with packages: `npm install`
-   To start server use : `php artisan serve`
-   Open localhost on port 8000
-   to start client use : `npm run dev`
